# Dashboard NBA Analytics
📝 **Funcionalidades**
    
* Visualização de estatísticas dos times
* Análise de desempenho de jogadores
* Gráficos interativos de performance
* Comparação entre times e jogadores
* Visualização de logos das equipes
* Análise de temporadas específicas

⚙️ **Ferramentas utilizadas**

* Python 3.8+
* Streamlit
* nba_api
* Pandas
* Plotly
* Matplotlib

📊 **Painéis Disponíveis**

1. **Visão Geral**
    - Métricas principais
    - Resumo da temporada
    - Classificação atual

2. **Análise de Times**
    - Estatísticas detalhadas
    - Histórico de jogos
    - Próximas partidas

3. **Análise de Jogadores**
    - Estatísticas individuais
    - Rankings
    - Comparações

4. **Gráficos**
    - Tendências de performance
    - Distribuição de pontuação
    - Análises avançadas
